Командная строка – это интерфейс взаимодействия с операционной системой, который позволяет пользователям выполнять различные операции и управлять компьютером с помощью текстовых команд. В отличие от графического интерфейса, командная строка предоставляет более гибкие и мощные возможности управления системой.

Запуск командной строки

Командную строку можно запустить на различных операционных системах. На Windows она называется "Командная строка" или "cmd", а на UNIX-подобных системах, таких как Linux и macOS, она известна как "Терминал" или "Консоль". Чтобы открыть командную строку на Windows, можно нажать комбинацию клавиш Win + R, ввести "cmd" и нажать Enter. На UNIX-подобных системах команду для запуска терминала можно найти в меню "Приложения" или использовать сочетание клавиш Ctrl + Alt + T.

Основные команды

  1. Команда CD

    2. Команда "cd" используется для перехода между каталогами. Например, "cd C:\Users" переместит вас в каталог "Users" на диске C:. Если вам нужно вернуться на уровень выше, вы можете использовать команду "cd ..". Команда "cd" без аргументов позволяет вернуться в домашний каталог.

  2. Команда DIR (и LS в UNIX-подобных системах)

    3. Команда "dir" (или "ls" в UNIX-подобных системах) используется для вывода списка файлов и каталогов в текущем каталоге. Например, "dir" выведет список файлов и каталогов в текущем каталоге.

  3. Команда MKDIR (или MD в Windows)

    4. Команда "mkdir" (или "md" в Windows) используется для создания нового каталога. Например, "mkdir NewFolder" создаст новый каталог с названием "NewFolder" в текущем каталоге.

  4. Команда RM (или DEL в Windows)

    5. Команда "rm" (или "del" в Windows) используется для удаления файлов. Например, "rm file.txt" удалит файл с названием "file.txt" из текущего каталога.

  5. Команда CP (или COPY в Windows)

    6. Команда "cp" (или "copy" в Windows) используется для копирования файлов. Например, "cp file.txt newfile.txt" создаст копию файла "file.txt" и назовет ее "newfile.txt" в текущем каталоге.

  6. Команда MV (или MOVE в Windows)

    7. Команда "mv" (или "move" в Windows) используется для перемещения или переименования файлов и каталогов. Например, "mv file.txt Documents" переместит файл "file.txt" в каталог "Documents".

  7. Команда PWD (или CD в Windows)

    8. Команда "pwd" (или "cd" в Windows) используется для вывода текущего каталога. Например, "pwd" выведет полный путь к текущему каталогу.

  8. Команда CLEAR (или CLS в Windows)

    9. Команда "clear" (или "cls" в Windows) используется для очистки экрана командной строки. Это помогает очистить вывод и улучшить читаемость.
