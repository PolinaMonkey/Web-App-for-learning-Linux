Управление пользователями и правами доступа является одной из важных задач в Linux. В этой главе мы рассмотрим основные аспекты управления пользователями, создания учетных записей, назначения прав доступа и управления группами пользователей. Мы также рассмотрим инструменты и команды, используемые для выполнения этих задач.

Учетные записи пользователей

Каждый пользователь в Linux имеет свою учетную запись, которая содержит информацию о пользователе, такую как имя пользователя, домашний каталог, идентификатор пользователя (UID) и другие связанные с ним значения. Учетная запись пользователя также может быть связана с группами пользователей.

  1. Создание учетных записей пользователей

    2. Для создания учетной записи пользователя в Linux используется команда "useradd". Например, чтобы создать нового пользователя с именем "user1", можно выполнить следующую команду:

    sudo useradd user1

    Эта команда создаст учетную запись пользователя "user1" с использованием настроек по умолчанию, включая домашний каталог пользователя и идентификатор пользователя. Однако, для более подробной настройки учетной записи пользователя, можно использовать различные опции команды "useradd".

  2. Изменение учетных записей пользователей

    3. Для изменения существующих учетных записей пользователей в Linux используется команда "usermod". Например, чтобы изменить имя пользователя "user1" на "user2", можно выполнить следующую команду:

    sudo usermod -l user2 user1

    Эта команда изменит имя пользователя "user1" на "user2". Аналогично, можно использовать различные опции команды "usermod" для изменения других параметров учетной записи пользователя.

  3. Удаление учетных записей пользователей

    4. Для удаления учетной записи пользователя в Linux используется команда "userdel". Например, чтобы удалить пользователя "user1", можно выполнить следующую команду:

    sudo userdel user1

    Эта команда удалит учетную запись пользователя "user1". Однако, она не удалит файлы и директории, связанные с этим пользователем. Для удаления всех файлов и директорий пользователя также можно использовать опцию "-r" команды "userdel".

Управление группами пользователей

Группы пользователей позволяют объединять пользователей для более удобного назначения прав доступа. В Linux каждый пользователь может находиться в одной или нескольких группах.

  1. Создание групп пользователей

    2. Для создания группы пользователей в Linux используется команда "groupadd". Например, чтобы создать новую группу с именем "group1", можно выполнить следующую команду:

    sudo groupadd group1

    Эта команда создаст новую группу с именем "group1". Как и с учетными записями пользователей, можно использовать различные опции команды "groupadd" для настройки создаваемой группы.

  2. Изменение групп пользователей

    3. Для изменения существующих групп пользователей в Linux используется команда "groupmod". Например, чтобы изменить имя группы "group1" на "group2", можно выполнить следующую команду:

    sudo groupmod -n group2 group1

    Эта команда изменит имя группы "group1" на "group2". Как и с учетными записями пользователей, можно использовать различные опции команды "groupmod" для изменения других параметров группы.

  3. Удаление групп пользователей

    4. Для удаления группы пользователей в Linux используется команда "groupdel". Например, чтобы удалить группу "group1", можно выполнить следующую команду:

    sudo groupdel group1

    Эта команда удалит группу "group1" и удалит все ссылки на эту группу из учетных записей пользователей.

Назначение прав доступа

Linux имеет гибкую систему прав доступа, которая позволяет устанавливать различные уровни доступа к файлам и директориям. Права доступа состоят из трех основных категорий: владелец, группа и другие пользователи.

  1. Права доступа к файлам и директориям

    2. Права доступа в Linux представлены в виде набора буквенных и числовых значений. Буквенные значения представляют базовые права доступа: чтение (r), запись (w) и выполнение (x). Числовые значения представляются трехзначными числами, которые представляют сумму базовых прав доступа (например, 750).

    • Чтение (r) позволяет просматривать содержимое файла или директории.
    • Запись (w) позволяет создавать, изменять или удалять файлы и директории.
    • Выполнение (x) позволяет выполнять файлы или входить в директории.
  2. Изменение прав доступа

    3. Для изменения прав доступа к файлам и директориям в Linux используется команда "chmod". Например, чтобы установить права доступа 755 для файла "file1.txt", можно выполнить следующую команду:

    sudo chmod 755 file1.txt

    Эта команда установит права доступа на чтение, запись и выполнение для владельца файла (7), чтение и выполнение для группы (5) и чтение и выполнение для других пользователей (5).
